1. 개요

JOO CRYPTO TRADING(이하 "앱")은 사용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 방침은 앱이 수집하는 정보, 사용 목적, 보관 방법에 대해 설명합니다.

2. 수집하는 정보

항목 수집 목적 보관 기간
이메일 주소 회원가입, 계정 인증 회원 탈퇴 시 즉시 삭제
비밀번호 (해시) 계정 보안 회원 탈퇴 시 즉시 삭제
거래소 API 키·시크릿 자동매매 서비스 제공 회원 탈퇴 시 즉시 삭제
거래 내역 수익 분석, 히스토리 조회 회원 탈퇴 시 즉시 삭제

3. 정보 이용 목적

  • 암호화폐 자동매매 서비스 제공
  • 거래 내역 및 수익 분석 제공
  • 계정 인증 및 보안 유지
  • 서비스 오류 진단 및 개선

4. 개인정보 보호 조치

  • 비밀번호: bcrypt 단방향 해시 암호화
  • 거래소 API 키: AES-256-GCM 양방향 암호화 후 저장
  • 모든 통신: HTTPS(TLS) 암호화
  • API 키는 거래 실행 시에만 일시적으로 복호화되며, 서버 메모리에만 존재

5. 제3자 제공

사용자의 개인정보는 다음의 경우를 제외하고 제3자에게 제공되지 않습니다:

  • 사용자가 직접 연결한 거래소(Binance 등)에 API 키를 통해 거래 요청을 전달하는 경우
  • 법령에 의해 요구되는 경우

6. 계정 및 데이터 삭제

사용자는 언제든지 앱 내에서 계정을 삭제할 수 있습니다:

앱 → 프로필 → 회원 탈퇴

탈퇴 즉시 이메일, API 키, 거래 내역 등 모든 데이터가 영구적으로 삭제됩니다.

7. 미성년자

본 앱은 만 19세 이상을 대상으로 합니다. 미성년자의 사용은 허용되지 않습니다.
